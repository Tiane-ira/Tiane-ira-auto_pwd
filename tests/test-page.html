选择器示例

CSS Selector示例：

  • #username - 通过ID选择
  • .email-input - 通过类名选择
  • input[name='password'] - 通过属性选择
  • form#testForm1 input - 组合选择器
  • button[type='submit'] - 按钮选择

XPath示例：

  • //input[@id='username'] - 通过ID选择
  • //input[@name='email'] - 通过name属性选择
  • //button[text()='提交'] - 通过文本内容选择
  • //select[@id='country'] - 选择下拉框
  • //form[@id='testForm1']//input - 组合XPath
