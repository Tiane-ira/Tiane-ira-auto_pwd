# 图标文件说明

插件需要以下图标文件：
- `icon16.png` (16x16像素)
- `icon48.png` (48x48像素)
- `icon128.png` (128x128像素)

## 快速生成图标

你可以使用以下方法之一来创建图标：

### 方法1：使用在线工具
访问 https://www.favicon-generator.org/ 或类似工具，上传你的图标并下载不同尺寸的版本。

### 方法2：使用图标生成HTML
打开项目根目录下的 `generate-icons.html` 文件，在浏览器中打开，然后下载生成的图标。

### 方法3：使用现有图片
如果你有现有的图标图片，可以使用图片编辑软件（如Photoshop、GIMP）或在线工具调整到所需尺寸。

## 临时解决方案

如果暂时没有图标文件，你可以：
1. 创建简单的纯色PNG图片作为占位符
2. 或者修改 `manifest.json`，暂时移除图标相关配置（插件仍可正常工作，只是没有图标显示）
